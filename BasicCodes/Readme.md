# Códigos Básicos, triângulo e formas geométricas
Só o básico

### Características

- Objetos 2D.

## Requisitos

Você precisará das seguintes dependências:

- Python 3.x
- Biblioteca OpenGL para Python

## Execução

Execute o script principal usando Python. Por exemplo:

```bash
python Simple_Triangulo.py
```
